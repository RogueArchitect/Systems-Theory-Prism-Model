Prism Model: A State‑Dependent Emergence Framework
Abstract
The Prism Model conceptualises creativity as an emergent property arising from the interaction between system boundaries, signal coherence, state‑dependent processing, and identity‑aligned transformation.
It provides a compressed, intuitive representation of how consciousness refracts incoming cognitive signals into novel creative output.

This framework sits at the intersection of systems theory, information processing, and emergent cognition.

System Architecture
1. Boundary Layer — Consciousness (The Prism)
Consciousness functions as a semi‑permeable boundary that filters and refracts incoming cognitive signals.
Its structure determines:

what enters

what is transformed

what is suppressed

This aligns with principles of system boundaries and identity‑preserving filters.

2. State Variable — Clarity (Bandwidth & Noise)
The “clarity” of the prism represents the system’s state:

Clear state → high bandwidth, low noise, high coherence

Murky state → low bandwidth, high noise, restricted access

Clarity is influenced by emotional regulation, environment, and cognitive load, consistent with state‑dependent cognition.

3. Input Layer — Thought Frequency (Signal)
Thoughts enter the system as frequencies:

emotional

symbolic

associative

conceptual

These signals form the raw material for creative transformation.

4. Transformation Layer — Refraction (Non‑Linear Recombination)
As signals pass through the prism, they undergo:

refraction

recombination

amplification

suppression

This is the system’s non‑linear transformation function, where new patterns begin to form.

5. Emergent Layer — Creative Output
Creativity is not stored anywhere in the system.
It emerges when:

boundaries are permeable

clarity is high

signals are coherent

identity alignment is maintained

This aligns with principles of emergent behaviour and complex adaptive cognition.
