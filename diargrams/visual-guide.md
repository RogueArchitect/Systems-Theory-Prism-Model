Clarity → State (bandwidth/noise)

Colours → Emergent creative patterns

3. Input Layer — Thought Frequency
Thoughts enter the system as signals:

emotional

symbolic

associative

conceptual

These signals behave like frequencies — some coherent, some noisy.

4. Boundary Layer — Consciousness
The prism (consciousness) acts as a semi‑permeable boundary:

filters signals

maintains identity alignment

regulates what gets transformed

A stable boundary = stable creativity.

5. State Variable — Clarity
Clarity determines how well signals pass through.

Clear State
high bandwidth

low noise

deep access

Murky State
low bandwidth

high noise

shallow access



6. Transformation Layer — Refraction
This is where creativity actually happens.

Signals are:

refracted

recombined

amplified

suppressed

This non‑linear transformation produces new patterns.



7. Emergent Layer — Creative Output
Creativity is not stored anywhere.
It emerges when:

clarity is high

boundaries are permeable

signals are coherent

The output is a new pattern that wasn’t present in the input.
