Creativity Walkthrough
A step‑by‑step example of the Prism Model in action.

1. Input Signal
A thought enters:
“Maybe I should design a new workflow.”

2. Prism State
Clarity is moderate — some noise, but manageable.

3. Refraction
The thought splits into:

workflow diagrams

user experience ideas

automation concepts

4. Emergence
A new idea forms:
“A modular workflow with adaptive branching.”

This is emergent behaviour — not present in the original signal.
